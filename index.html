Font Properties

Property Results Example
font-family Specify the font family for text. font-family: Arial, sans-serif;
font-size Set the size of the font. font-size: 16px;
font-weight Control the thickness of the font. font-weight: bold;
font-style Define the style of the font (italic, normal). font-style: italic;
font-variant Control the variant of the font (normal, small-caps). font-variant: normal;

Text Properties

Property Results Example
Color Set the color of the text. color: #333;
line-height Specify the height of each line of text. line-height: 1.5;
text-align Align text within its container. text-align: center;
text-decoration Add decoration to text (underline, overline, line-through). text-decoration: underline;
text-transform Transform text to uppercase, lowercase, or capitalize. text-transform: uppercase;

Property Results Example
